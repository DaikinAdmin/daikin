## Test Execution Flow

```
1. Global Setup (global-setup.ts)
   └─> Authenticate all roles
       └─> Save states to .auth/{ROLE}.json

2. Test Starts
   └─> Fixture loads appropriate role context
       └─> Creates Application instance
           └─> Initializes all page objects
               └─> Test uses page objects to interact
                   └─> Actions wrapped in test.step()
                       └─> Assertions verify behavior
```
