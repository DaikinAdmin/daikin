## Key UI Improvements

### Product Images
1. **Visual Upload Area**: Large, clickable dashed border area
2. **Preview Grid**: 2-3 column responsive grid of thumbnails
3. **Progress Indicator**: Shows "Uploading... (2/3)" during upload
4. **Hover Effects**: X button appears on hover over images
5. **Counter**: Shows "Images (2/5)" to track limit
6. **Table Preview**: Shows small thumbnails in the table
7. **Error Display**: Red banner with clear error message

### Product Items
1. **Large Upload Area**: Full-width dashed border for single image
2. **Large Preview**: Aspect-video ratio for better visibility
3. **Hover Remove**: X button appears on hover
4. **Per-Item Errors**: Error message above upload area
5. **Loading State**: Spinner with "Uploading..." text
6. **Replace Flow**: Click upload area again to change image

### Interaction Patterns

**Multi-Select Flow (Product Images):**
1. Click upload area
2. File picker opens (multi-select enabled)
3. Select 1-5 images
4. Upload begins automatically
5. Progress shows for each file
6. Thumbnails appear as uploaded
7. Hover over any to remove

**Single-Select Flow (Product Items):**
1. Click upload area
2. File picker opens (single file)
3. Select 1 image
4. Upload begins automatically
5. Large preview appears
6. Hover to show remove button
7. Click upload area again to replace
